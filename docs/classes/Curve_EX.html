tension

tension: number

Inherited from pc.Curve.tension

  • Defined in node_modules/playcanvas/build/playcanvas.d.ts:17025

Controls how {@link CURVE_SPLINE} tangents are calculated. Valid range is between 0 and 1 where 0 results in a non-smooth curve (equivalent to linear interpolation) and 1 results in a very smooth curve. Use 0.5 for a Catmull-rom spline.

type

type: number

Inherited from pc.Curve.type

  • Defined in node_modules/playcanvas/build/playcanvas.d.ts:17017

The curve interpolation scheme. Can be:

  • {@link CURVE_LINEAR}
  • {@link CURVE_SMOOTHSTEP}
  • {@link CURVE_SPLINE}
  • {@link CURVE_STEP}

Defaults to {@link CURVE_SMOOTHSTEP}.
